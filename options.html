Кабинет с настройками кнопки и принимающего телефона здесь.

Сервис состоит из двух частей: клиент и оператор.

Клиент инициирует звонок, оператор принимает звонок.

Клиент - кнопка "Звонок с сайта"
Оператор - WEB телефон
Эту страницу можно всегда вызвать кликнув на иконку    в правом вернем углу Chrome

Основная парадигма WebPhone принимать звонки с кнопки.
WebPhone - это встроенный в браузер SIP телефон позволяющий принимать и отправлять звонки на SIP, фиксированные и мобильные номера.

Необходимые условия для включения телефона:

  • Микрофон разрешен
  • Телефон зарегистрирован

    Варианты применения WebPhone:
Бесплатно подробнее...

  • Принимать звонки с кнопки "Callme" расположенной на вашем сайте
    Кнопка -> WebPhone
  • ТПринимать звонки с мобильных, стационарных, SIP телефонов
    GSM, PSTN, SIP -> WebPhone
  • Принимать звонки с мобильных без руоминга.
    Необходимо переадресовать звонки со своего мобильного на наш номер (067, 050, 063)
    GSM -> WebPhone

Платно подробнее...

  • Звонить на мобильные, стационарные, SIP телефоны
  • Пользоваться услугами колл-центра
  • Карточка клиента
  • Статистика входящих звонков
  • Статистика исходящих звонков
  • Запись разговоров
  • Голосовая почта
  • 1С бухгалтерия интерфейс
